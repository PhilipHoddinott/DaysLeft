# Days Left Countdown

A simple static web app that counts down to a selected date and time. The default target is July 31, 2026 at noon.

## Features
- Shows days left
- Shows total hours left
- Shows total minutes left
- Pure HTML, CSS, and JavaScript
- Ready for GitHub Pages deployment

## Usage
Open `index.html` in your browser and adjust the target date/time if needed.
